# Guard Villagers (Fabric)

Better village pest control — armed guards that defend villages, plus villager AI tweaks.

This repository is a **Fabric** fork maintained for continued development and integration with other Fabric mods.

## Lineage

This mod has two upstreams:

| Role | Project | Links |
|------|---------|-------|
| **Fork parent** (Fabric port) | MrSterner's GuardVillagers | [GitHub — mrsterner/GuardVillagers](https://github.com/mrsterner/GuardVillagers) |
| **Original mod** (Forge / NeoForge) | Guard Villagers by seymourimadeit | [Modrinth — guard-villagers](https://modrinth.com/mod/guard-villagers) · [GitHub — seymourimadeit/guardvillagers](https://github.com/seymourimadeit/guardvillagers) |

**This fork:** [ImpureCrumpet/mc-guardvillagers](https://github.com/ImpureCrumpet/mc-guardvillagers)

Gameplay and design credit belong to the original Forge/NeoForge mod. The Fabric port and subsequent forks carry that work forward on Fabric Loader.

## Requirements

- Minecraft **1.20.1**
- [Fabric Loader](https://fabricmc.net/use/)
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [MidnightLib](https://modrinth.com/mod/midnightlib) (config UI)

## Building

```bash
./gradlew build
```

The built JAR is in `build/libs/`.
